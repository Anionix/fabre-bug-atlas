# Runbooks

Runbooks describe reusable operating flows.

## Files

- [`bug-triage.md`](bug-triage.md): classify and fix review bugs.
- [`public-evidence.md`](public-evidence.md): keep public review text safe.
- [`review-readback.md`](review-readback.md): read GitHub PR/review state
  without pasting full transcripts.
- [`token-efficient-closeout.md`](token-efficient-closeout.md): compress
  successful validation and expand only failures.

## Usage

Copy the smallest useful section into the consumer repository. Keep
repo-specific commands in the consumer repository, not in this atlas.
