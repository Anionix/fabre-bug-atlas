# Security Policy

This repository is public. Do not report secrets by opening public issues or
pull requests.

## If You Find Sensitive Data

If you find a credential, private path, private repository link, or raw
environment value in this repository, report it privately to the repository
owner using GitHub's private vulnerability reporting when available, or through
a trusted private channel.

## What To Avoid Publishing

- API keys, tokens, or credential-shaped strings.
- Local absolute paths.
- Raw environment values.
- Private repository URLs.
- Raw logs from private systems.

## Normal Review Bugs

Non-sensitive wording, taxonomy, template, and runbook issues can be reported
with public issues or pull requests.
