# Templates

Templates are public-safe starting points for review and closeout comments.

## Files

- [`fix-comment.md`](fix-comment.md): reply to a review finding after a fix.
- [`pr-closeout.md`](pr-closeout.md): summarize a pull request before review.
- [`issue-closeout.md`](issue-closeout.md): close an issue with merged evidence.
- [`merge-stack-readback.md`](merge-stack-readback.md): summarize stacked PR
  state.

## Usage

Replace placeholders before publishing. Scan the exact text that will be
posted, and scan the patch separately before claiming the patch is clean.
